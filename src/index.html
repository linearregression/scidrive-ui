Frequently Asked Questions

How is the space allocated per user?

The suggested user quota is 1TB, not enforced. If more space is needed, it is recommended that you contact us to explain the purpose of using the storage.

What is the maximum supported file size?

The maximum file size is 5GB. Larger files can be uploaded in segmented mode using the Python client. Once uploaded, the file can be downloaded as a single file. The size for segmented uploads is not limited.

What is the maximum supported number of files?

The number of files is not limited. Single container can contain millions of files.

How is the data kept private? Will somebody be able to see my data after I upload it?

The data will be available to the owner only until some part of it is shared and the link is given to someone. The metadata extracted and uploaded to a 3rd party service will be handled according to the service policy.

The system searches for metadata in files, but this is only workable for reasonably standard-compliant and complete self-documenting formats. How does your system handle unknown file types?

A special plugin can be developed to handle such data if needed, otherwise nothing will be extracted from these files. You can limit the metadata extraction to containers where you have files in standard formats.
